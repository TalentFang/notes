## 1. 概述

文献计量学（Biblio metrics/Research Metrics）是用数学和统计学方法，对学术文献、专利、期刊等学术成果进行定量分析研究的学科。

### 发展历程

| 时间 | 里程碑 |
|------|--------|
| 1917 年 | Cole 和 Eales 首次对文献进行统计分析 |
| 1926 年 | Lotka 提出文献计量学第一定律（平方反比定律） |
| 1934 年 | Bradford 发现文献分散定律 |
| 1955 年 | Garfield 提出 SCI 概念 |
| 1969 年 | Alan Pritchard 正式提出"文献计量学"术语 |
| 2004 年 | H-index 被提出 |

### 应用领域

- 评估科研机构/个人的学术影响力
- 分析学科发展趋势和研究热点
- 评价期刊学术水平
- 规划科研决策
- 预测技术发展方向

## 2. 核心概念

### 2.1 基本计量指标

#### 发文量 (Publication Count)

衡量科研产出的最直接指标。

```
发文量 = 统计范围内论文总数
```

#### 被引频次 (Citation Count)

论文被引用的次数，是衡量学术影响力的重要指标。

```
总被引频次 = 论文A被引次数 + 论文B被引次数 + ...
```

#### 篇均被引 (Citations Per Paper)

```
篇均被引 = 总被引频次 / 论文总数
```

#### H-index (H 指数)

H指数的含义：某作者或机构有 H 篇论文每篇至少被引用 H 次。

**计算示例：**

| 论文 | 被引次数 | 是否计入 H-index |
|------|----------|------------------|
| A | 50 | ✅ (≥5) |
| B | 30 | ✅ (≥5) |
| C | 20 | ✅ (≥5) |
| D | 8 | ✅ (≥5) |
| E | 5 | ✅ (≥5) |
| F | 3 | ❌ (<5) |
| G | 1 | ❌ (<5) |

H-index = 5（因为有5篇论文每篇至少被引用5次）

#### G-index

G 指数：前 G 篇论文的总被引次数不小于 G²，且 G 是满足这一条件的最大整数。

**计算示例：**

| 论文（按被引降序） | 被引次数 | 累计被引 | G² |
|-------------------|----------|----------|-----|
| A | 100 | 100 | 1 |
| B | 60 | 160 | 4 |
| C | 40 | 200 | 9 |
| D | 25 | 225 | 16 |
| E | 15 | 240 | 25 |

G = 4（因为前4篇累计240次 ≥ 16，且前5篇累计255次 < 25）

#### I10-index

Google Scholar 使用的指标，表示被引用至少10次的论文数量。

#### 期刊影响因子 (Impact Factor, IF)

```
期刊影响因子 = 该刊前两年发表论文在当年的被引总次数 / 该刊前两年发表论文总数
```

**示例：**
- 2023 年某期刊影响因子 = (2021+2022年论文在2023年被引次数) / (2021+2022年发表论文总数)

| 指标 | 说明 |
|------|------|
| 影响因子 (IF) | 期刊两年内论文平均被引次数 |

## 3. 基本方法

### 3.1 经典文献计量定律

#### Lotka 定律（平方反比定律）

描述作者与发文量关系的定律。

```
f(x) = C / x²
```

其中 x 为发文数，f(x) 为发表 x 篇论文的作者占比，C 为常数。

**含义：** 发文 n 篇的作者数约为发 1 篇作者数的 1/n²。

#### Bradford 定律（分散定律）

描述文献在期刊中分布的定律。

核心结论：**少量期刊包含大量相关文献，大量期刊包含少量相关文献。**

**布拉德福分区：**
- 核心区：包含 1/3 论文的期刊数量
- 相关区：包含 1/3 论文的期刊数量
- 边缘区：包含 1/3 论文的期刊数量

比例关系约为：1 : n : n²

#### Zipf 定律

描述词频分布的定律。

```
r × f(r) = C
```

其中 r 为词频排名，f(r) 为词频，C 为常数。

## 6. 注意事项

### 6.1 数据质量

1. **检索式设计**：检索式决定样本的完整性和相关性
2. **去重处理**：同一论文可能被多个数据库收录
3. **字段标准化**：作者名、机构名可能有多种写法
4. **时间窗口**：选择合理的时间范围

### 6.2 指标局限性

| 指标 | 局限 |
|------|------|
| 发文量 | 不反映研究质量 |
| 被引次数 | 随时间累积，有学科差异 |
| H-index | 受学科、年龄影响 |
| IF | 期刊级别≠论文级别 |

### 6.3 常见问题

**Q: 不同数据库数据不一致怎么办？**
A: 建议以一个主要数据库为主，交叉验证时注意去重。

**Q: 如何处理重名作者？**
A: 使用 ORCID 等唯一标识符，或结合机构信息判断。

**Q: 自引是否应该剔除？**
A: 根据研究目的决定，一般可同时报告含自引和不含自引的数据。

**Q: 如何选择研究热点识别方法？**
A: 高频词分析快速直观，突现词分析反映新兴趋势，共词聚类揭示主题结构。

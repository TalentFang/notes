## 运行模型
```bash
# 安装运行
ollama run deepseek-coder-v2:16b
# ollama pull carstenuhlig/omnicoder-9b:q4_k_m  

# 或拉取量化版本（更省显存）
ollama pull deepseek-coder-v2:16b-lite-q4_0
```

Ollama 的基本使用非常直观，核心就是“安装-拉取-运行”这三步，让你能在自己的电脑上轻松跑起大模型。

下面是它的核心使用流程和一些常用操作的速查表。
### ⚙️ 核心命令速查

| 类别       | 命令示例                | 作用说明                              |
| :------- | :------------------ | :-------------------------------- |
| **模型操作** | `ollama pull <模型名>` | 从官方仓库**下载**模型到本地。                 |
|          | `ollama run <模型名>`  | **运行**模型，进入交互式对话。如果本地没有，会自动下载。    |
|          | `ollama list`       | 查看所有已下载到本地的模型列表。                  |
|          | `ollama ps`         | 查看当前正在运行的模型及其状态。                  |
|          | `ollama stop <模型名>` | 停止一个正在运行的模型，以释放内存资源。              |
|          | `ollama rm <模型名>`   | 删除本地的一个模型，以释放磁盘空间。                |
| **服务管理** | `ollama serve`      | 启动 Ollama 的后台服务。通常情况下，安装后服务会自动运行。 |
| **信息查看** | `ollama show <模型名>` | 显示模型的详细信息，如参数、模板等。                |
|          | `ollama --version`  | 查看已安装的 Ollama 版本号。                |

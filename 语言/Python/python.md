## 概述

本文档梳理 Python 开发中的常见用法，重点关注工程实践而非基础语法。

### 与 Java 的主要区别

| 特性 | Python | Java |
|------|--------|------|
| 类型系统 | 动态类型 + 类型注解 | 静态类型 (强类型) |
| 运行方式 | 解释执行 / .pyc | 编译后运行 .class |
| 包管理 | pip / poetry / conda | Maven / Gradle |
| 虚拟环境 | venv / virtualenv | 无 (Maven 管理依赖) |
| 多版本共存 | 需工具 (pyenv) | 同时安装多版本 |

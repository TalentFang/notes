# 模块与导入

## 模块基本概念

Python 中每个 `.py` 文件就是一个模块。模块是代码组织的基本单位。

```python
# my_module.py
def greet(name):
    return f"Hello, {name}"

class MyClass:
    pass

# 模块级变量
VERSION = "1.0"
```

## 导入语法

### 基本导入

```python
# 导入整个模块
import my_module
my_module.greet("World")

# 导入特定成员
from my_module import greet, MyClass
greet("World")

# 导入并起别名
from my_module import greet as hello
hello("World")

# 为模块起别名
import my_module as mm
mm.greet("World")

# 导入所有 (不推荐)
from my_module import *
```

## `__init__.py` - 包初始化

```
my_package/
├── __init__.py      # 包初始化，可设置 __all__、导入子模块
├── module1.py
├── module2.py
└── sub_package/
    ├── __init__.py
    └── module3.py
```

### `__init__.py` 作用

```python
# my_package/__init__.py
# 方式1: 指定导出
from .module1 import ClassA, function_b

# 方式2: 定义 __all__
__all__ = ['ClassA', 'function_b']

# 方式3: 批量导入子模块
from . import module1
from . import module2
```

### 导入顺序

```
import my_package          # 触发 my_package/__init__.py
my_package.module1        # 现在才加载 module1
```

## 相对导入与绝对导入

```python
# 绝对导入 (推荐)
from my_package.module1 import ClassA

# 相对导入 (包内部使用)
from . import module1           # 同级
from .. import sibling_module   # 上级
from .sub_package import mod   # 下级
```

## 模块搜索路径

```python
import sys
print(sys.path)

# 添加搜索路径
sys.path.insert(0, '/path/to/add')
```

**搜索顺序**:
1. 当前目录
2. `PYTHONPATH` 环境变量
3. 安装目录 (`site-packages` 等)

## 常见模式

### 单例模式

```python
# config.py
class _Config:
    def __init__(self):
        self.debug = True
        self.db_host = "localhost"

_config = _Config()

def get_config():
    return _config
```

### 延迟导入

```python
# 避免循环导入，或加速启动
def heavy_function():
    import heavy_module  # 仅在调用时导入
    return heavy_module.do_something()
```

### 可选依赖

```python
# 只在需要时才导入，没有时给出提示
try:
    import pandas as pd
    HAS_PANDAS = True
except ImportError:
    HAS_PANDAS = False
    pd = None
```

## 与 Java 对比

| Python | Java |
|--------|------|
| `import my_module` | `import com.example.MyClass` |
| `from pkg import mod` | `import com.example.sub.*` |
| `__init__.py` | `package` 声明 |
| 无对应 | `static import` |

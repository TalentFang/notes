## 常用库分类

### 网络请求

```python
# requests - HTTP 请求
import requests

response = requests.get("https://api.example.com/data")
response.raise_for_status()  # 抛出异常而不是无声失败
data = response.json()        # 解析 JSON

# POST 请求
response = requests.post(
    "https://api.example.com/submit",
    json={"key": "value"},   # 自动 JSON 序列化
    headers={"Authorization": "Bearer token"}
)
```

### 数据库

## 调用方式总结

import json

## 动态导入模式

```python
# 字符串转模块
def load_db_module(db_type):
    """根据配置动态加载数据库模块"""
    if db_type == "mysql":
        import pymysql
        return pymysql
    elif db_type in ("postgresql", "vastbase"):
        import psycopg2
        return psycopg2
    elif db_type == "dm":
        import dmPython
        return dmPython
    else:
        raise ValueError(f"Unsupported db_type: {db_type}")

# 使用
db_module = load_db_module("vastbase")
conn = db_module.connect(host="localhost", port=5432, ...)
```

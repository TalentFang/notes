# PostgreSQL 数据类型

## 数值类型

### 整数
| 类型 | 存储 | 范围 |
|---|---|---|
| SMALLINT (INT2) | 2字节 | -32768 ~ 32767 |
| INTEGER (INT4) | 4字节 | -2147483648 ~ 2147483647 |
| BIGINT (INT8) | 8字节 | -9223372036854775808 ~ 9223372036854775807 |
| SMALLSERIAL | 2字节 | 1 ~ 32767 |
| SERIAL | 4字节 | 1 ~ 2147483647 |
| BIGSERIAL | 8字节 | 1 ~ 9223372036854775807 |

```sql
CREATE TABLE orders (
    id SERIAL PRIMARY KEY,
    order_no BIGINT
);
```

### 浮点数
| 类型 | 存储 | 说明 |
|---|---|---|
| REAL (FLOAT4) | 4字节 | 6位精度 |
| DOUBLE PRECISION (FLOAT8) | 8字节 | 15位精度 |
| DECIMAL(p,s) | 可变 | 定点数，p=总位数，s=小数位 |
| NUMERIC(p,s) | 可变 | 同 DECIMAL |

```sql
price DECIMAL(10, 2)  -- 99999999.99
```

## 字符串类型

| 类型 | 说明 |
|---|---|
| CHAR(n) | 定长，空白补齐 |
| VARCHAR(n) | 变长，最大 n 字符 |
| TEXT | 变长，无限长度 |

```sql
name CHAR(10)      -- 不足补空格
name VARCHAR(100)   -- 实际长度
content TEXT       -- 无长度限制
```

## 日期时间类型

| 类型 | 说明 | 示例 |
|---|---|---|
| DATE | 日期 | 2024-01-15 |
| TIME | 时间 | 13:45:30 |
| TIMESTAMP | 日期时间（无时区） | 2024-01-15 13:45:30 |
| TIMESTAMPTZ | 日期时间（有时区） | 2024-01-15 13:45:30+08 |
| INTERVAL | 时间间隔 | 3 days |

```sql
created_at TIMESTAMP DEFAULT NOW();
updated_at TIMESTAMPTZ DEFAULT NOW();
```

## 布尔类型

```sql
is_active BOOLEAN  -- TRUE / FALSE / NULL
-- 字符串输入：'true', 'false', 'yes', 'no', '1', '0'
```

## JSON 类型

| 类型 | 说明 |
|---|---|
| JSON | 验证 JSON 格式，存储为文本 |
| JSONB | 二进制格式，支持索引 |

```sql
-- JSONB 更适合需要查询的场景
CREATE TABLE events (
    id SERIAL,
    data JSONB
);

-- 查询 JSON 字段
SELECT data->>'name' FROM events;
SELECT data->'age' FROM events WHERE data->>'name' = '张三';
```

## 数组类型

```sql
-- 创建数组列
CREATE TABLE scores (
    student VARCHAR(50),
    scores INTEGER[]
);

-- 插入数组
INSERT INTO scores VALUES ('张三', ARRAY[90, 85, 88]);

-- 查询数组
SELECT * FROM scores WHERE 90 = ANY(scores);
```

## 范围类型

```sql
-- 日期范围
CREATE TABLE reservations (
    room VARCHAR(20),
    stay daterange
);

-- 查询范围包含
SELECT * FROM reservations WHERE daterange '[2024-01-01,2024-01-05)' @> '2024-01-03';
```

## UUID 类型

```sql
-- 生成 UUID
SELECT gen_random_uuid();

-- 存储 UUID
CREATE TABLE sessions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    token VARCHAR(255)
);
```

## 网络地址类型

| 类型 | 说明 |
|---|---|
| INET | IPv4 或 IPv6 地址 + 子网掩码 |
| CIDR | 网络地址（无主机部分） |
| MACADDR | MAC 地址 |

```sql
ip INET
netmask CIDR
```

# PostgreSQL 入门指南

## 概述

PostgreSQL 是功能强大的开源对象关系型数据库系统，以稳定性、性能和功能丰富著称。

## 适用场景

- 复杂业务逻辑
- GIS 应用（PostGIS 扩展）
- 高并发读写
- 需要事务完整性
- JSON/JSONB 支持

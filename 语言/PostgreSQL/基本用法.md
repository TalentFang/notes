# PostgreSQL 基本用法

## 连接数据库

```bash
# 基本连接
psql -h localhost -p 5432 -U username -d database

# 使用环境变量
export PGPASSWORD=password
psql -h localhost -U username -d database

# 连接字符串（URI）
psql postgresql://username:password@localhost:5432/database
```

## 数据库操作

```sql
-- 查看所有数据库
\l
SELECT datname FROM pg_database;

-- 创建数据库
CREATE DATABASE mydb;

-- 删除数据库
DROP DATABASE mydb;

-- 切换数据库
\c mydb

-- 查看当前数据库
SELECT current_database();
```

## Schema 操作

```sql
-- 查看所有 schema
\dn

-- 创建 schema
CREATE SCHEMA myschema;

-- 删除 schema
DROP SCHEMA myschema;

-- 设置 search path
SET search_path TO myschema, public;
```

## 表操作

```sql
-- 创建表
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(255) UNIQUE,
    age INTEGER DEFAULT 18,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 查看表结构
\d users

-- 查看所有表
\dt

-- 删除表
DROP TABLE users;

-- 清空表
TRUNCATE TABLE users;
```

## 索引操作

```sql
-- 创建索引
CREATE INDEX idx_name ON users(email);

-- 创建唯一索引
CREATE UNIQUE INDEX idx_unique_email ON users(email);

-- 查看索引
\d users

-- 删除索引
DROP INDEX idx_name;
```

## 数据操作（CRUD）

```sql
-- 插入数据
INSERT INTO users (name, email) VALUES ('张三', 'zhang@example.com');

-- 批量插入
INSERT INTO users (name, email) VALUES
    ('李四', 'li@example.com'),
    ('王五', 'wang@example.com');

-- 查询
SELECT * FROM users WHERE name = '张三';

-- 更新
UPDATE users SET age = 20 WHERE name = '张三';

-- 删除
DELETE FROM users WHERE id = 1;
```

## 用户/角色操作

```sql
-- 创建角色
CREATE ROLE readonly LOGIN PASSWORD '123456';

-- 创建用户（带登录权限）
CREATE USER admin WITH PASSWORD '123456';

-- 授权
GRANT SELECT ON users TO readonly;

-- 撤销权限
REVOKE SELECT ON users FROM readonly;

-- 查看所有角色
\du
```

## 常用函数

```sql
-- 当前时间
SELECT NOW();

-- 字符串操作
SELECT UPPER('hello');
SELECT LENGTH('hello');
SELECT SUBSTRING('hello world', 1, 5);

-- 聚合函数
SELECT COUNT(*) FROM users;
SELECT AVG(age) FROM users;
SELECT SUM(age) FROM users;
```

## 备份与恢复

```bash
# 备份单个数据库
pg_dump -h localhost -U username -d mydb > backup.sql

# 恢复
psql -h localhost -U username -d mydb < backup.sql

# 备份所有数据库
pg_dumpall -h localhost -U username > all_backup.sql
```

## psql 常用命令

| 命令 | 说明 |
|---|---|
| `\l` | 列出所有数据库 |
| `\c dbname` | 切换数据库 |
| `\d` | 列出当前数据库所有表 |
| `\d tablename` | 查看表结构 |
| `\dn` | 列出所有 schema |
| `\du` | 列出所有角色 |
| `\di` | 列出所有索引 |
| `\df` | 列出所有函数 |
| `\x` | 切换展开/压缩输出 |
| `\h sql` | 查看 SQL 语法帮助 |
| `\q` | 退出 psql |
